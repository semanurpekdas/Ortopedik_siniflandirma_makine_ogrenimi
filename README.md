# Ortopedik Sınıflandırma – Makine Öğrenimi

Bu proje, ortopedik veriler kullanılarak makine öğrenimi yöntemleri ile sınıflandırma yapılmasını amaçlamaktadır.  
Amaç, hastalara ait ölçümlerden ortopedik rahatsızlıkların doğru şekilde tahmin edilmesidir.

## Kullanılan Yöntemler

- Veri Ön İşleme
  - Eksik verilerin temizlenmesi
  - Normalizasyon işlemleri
- Özellik Seçimi
- Makine Öğrenimi Algoritmaları
  - Logistic Regression
  - KNN
  - SVM
  - Random Forest

## Kullanılan Veri Seti
Veri Türü: Ortopedik klinik veriler
Özellikler: (örnek)
Pelvik eğim
Lomber açı
Sakral eğim
Hedef Değişken: Ortopedik sınıf etiketi

Python
